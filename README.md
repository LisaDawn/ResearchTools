Qualitative Coder (QC) is a simple web page which enables you to upload a transcript and code it with tags you've designed to enable easier retrieval through classification of text with those codes.

LOADING TRANSCRIPTS 
=============================
You being by uploading a previous transcript ("Load state") or by pasting into the window found in the "Paste" tab. By starring your transcript with an identifier, you can differentiate text from multiple transcripts in any final reports.


ENCODING TRANSCRIPTS
==============================
Once transcript is loaded, in the "Code" tab, you can highlight text by clicking on the beginning and end of text to be highlighted, then assign a code in the sidebar through clicking it. Note that codes can be added, removed, and edited. 
There will be simple summary stat next to each code to show how many times it's been used in this state. There is also a summary stat at the top of the sidebar to show which transcripts have been loaded and how many codes they have attached to each transcript.

SAVING STATE 
======================
Note that you can save your state at any time by clicking "Save state". This will enable you to upload your current state to another QC instance at another time. This allows you to shut down your browser and not lose your work.

REPORTING 
===================
You can also "Export" your state which will give you a .txt file with this format:

-- TRANSCRIPT: PARTICIPANT-ID --
[CODE: QUOTE]

for each transcript. 

-- CROSS-TRANSCRIPT CODE SUMMARY --
## CODE
number of instances . number of participants: PARTICIPANT-ID
[NAME] "QUOTES"

The "Report" tab shows stats at the top: the number of transcripts, the number of annotations, and the number of codes used. The Report is cross-transcript color-coded by code.
